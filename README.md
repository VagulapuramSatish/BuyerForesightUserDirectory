📊 ***User Directory Dashboard***

A simple and interactive User Directory Dashboard that displays a list of users in a card layout. Clicking on a user card reveals detailed information about the selected user.

🚀 ***Features***

- 📁 Display users in a clean card-based layout
- 👆 Click on a user card to view detailed user information
- 🔍 Easy navigation and intuitive UI
- ⚡ Fast and responsive design
- 🧩 Modular and scalable structure

🖼️ ***Preview***

## User List View

- Displays all users as cards with basic information (e.g., name, avatar)
  
## User Detail View

- Shows detailed information when a card is clicked:
- Full Name
- Email
- Phone Number
- Address
- Other relevant details
